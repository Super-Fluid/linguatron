Linguatron

Thanks for helping with my linguistics 229 final project! It's a system that lets you store text along with symbols that you create and recall the text by drawing the same symbols. It's designed as a way to communicate that's faster than typing, to be used by people who have trouble speaking.

In order for the algorithm to work well, I need some training data: a set of symbols that people create and later attempts to draw the same symbols. That's where you can help! The next page will ask you to draw some symbols that come to mind when you think of certain words. Then it will ask you to recreate the same symbols from memory a few times. The whole process will take TODO minutes.

Please try to use symbols that make sense to you. Keep them simple and fast.
